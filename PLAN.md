# Техническое задание

## Проект

Разработать полностью адаптивный дизайн коммерческой платформы нового поколения, сочетающей возможности **TikTok + интернет-магазина + маркетплейса + прямых эфиров (Live Shopping)**.

Проект предназначен для:

* покупателей;
* продавцов;
* блогеров;
* компаний;
* администраторов.

Главная идея — пользователь может смотреть короткие видео, прямые эфиры и сразу покупать товары, не покидая приложение.

---

# Основные требования

Создать полноценный современный UI/UX.

Никаких заглушек.

Все экраны должны выглядеть как готовый коммерческий продукт.

---

# Итоговый результат

Claude Design должен разработать полноценный коммерческий дизайн-проект уровня мировых платформ, включающий:

* Полную дизайн-систему (Design System).
* UI Kit со всеми компонентами.
* Все экраны для покупателя.
* Все экраны для продавца.
* Все экраны администратора.
* Все состояния интерфейса (пустые, загрузка, ошибки, успех).
* Светлую и темную темы.
* Полностью адаптивные версии для Desktop, Laptop, Tablet и Mobile.
* Прототип пользовательских сценариев (User Flow).
* Современный премиальный интерфейс, готовый для реализации в **Figma** или последующей разработке фронтенда.
